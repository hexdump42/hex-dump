Detecting HTML5 Support

Modernizr (http://www.modernizr.com/) is an open source JavaScript library that detects the availability of native implementations of many HTML5 & CSS3 features

					<!DOCTYPE html>
					<html>
					<head>
					  <script src="modernizr.min.js"></script>
					</head>
					<body>
					  <script>if (Modernizr.canvas) { doSomething();}</script>
					</body>
					</html>
