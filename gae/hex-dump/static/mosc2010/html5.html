2D Drawing Canvas

SVG

				<svg>
				<circle id="aCircle" class="important" cx="50%" cy="50%" r="100"
				onmousedown="alert('you clicked on a circle');"/>
				</svg>

Not all browsers support inline SVG, but javascript created SVG will work in most HTML5 browsers

				<script>
					var container = document.getElementById('svg-demo');
					var svgns = 'http://www.w3.org/2000/svg';
					var svg = document.createElementNS(svgns, 'svg');
					svg.setAttribute('width', '100px');svg.setAttribute('height', '100px');
					var circle = document.createElementNS(svgns, 'circle');
					circle.setAttribute('r', 48);circle.setAttribute('fill','blue');
					circle.setAttribute('cx', '50%');circle.setAttribute('cy', '50%');
					svg.appendChild(circle);container.appendChild(svg);
				</script>

    Local Storage

    Web Storage specifies two ways to do "local" persistent data.

    sessionStorage is used to save data during lifetime of open tab or window

    localStorage is used to store data for longer periods, and survives browser close

    Detecting HTML5 Support

    Modernizr (http://www.modernizr.com/) is an open source JavaScript library that detects the availability of native implementations of many HTML5 & CSS3 features

    					<!DOCTYPE html>
					<html>
					<head>
					  <script src="modernizr.min.js"></script>
					</head>
					<body>
					  <script>if (Modernizr.canvas) { doSomething();}</script>
					</body>
					</html>
